## How to do it

Download this code.
Create a repository named `prajakta731.github.io`

Open CMD in downloaded folder

All commands will be available there

replace `git add README.md` to `git add .` everything else can be same

after it's done refresh and see all the code in the repository

now go to repository settings -> pages -> select branch `main` and root folder `/`.

and then click on choose theme  and select the default and click select theme and done in few minutes you can see the site.
